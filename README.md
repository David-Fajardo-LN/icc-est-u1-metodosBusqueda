DESCRIPCION:
El programa pide el tamaño del arreglo con el cual se va a trabajar, en base a este dato, usa un bucle for para generar el arreglo.
Despues usa el metodo de ordenamiento shell para ordenar el arreglo segun la edad, en orden ascendente.
Una vez ordenado, pide la edad a buscar y devuelve la primera coincidencia.
